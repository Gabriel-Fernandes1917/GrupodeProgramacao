Espaçamento



Salve guys, neste poster iremos falar sobre os tipos de espaçamento no HTML e CSS como padding, margin, width e height. Entre suas principais utilidades esta determinar um espaçamento interno, externo, maximo ou minimo de um elemento. Logo abaixo iremos ver alguns deles e exemplo dos mesmos.



Margin: Ele é usado para dar espaçamento externo de um elemento como uma div por exemplo. Vejamos um exemplo logo a baixo.



HTML

<div class="margin">
<p>

Elmento

</p>
</div>



Sem margin

Elmento sem margin



CSS

.margin > p{
margin-left: 100px;
}



Com margin

Elmento com margin



Como podemos ver com o uso do margin-left(margem-esquerda) nos conseguimos dar um espaçamento externo de 100px. Agora você deve estar se perguntando, é como se faz isso para a direita, cima e baixo ? Bem para dar margem para a esquerda vimos que é margin-left, então para a direita é margin-right(margem-direita), cima é margin-top(margem-topo) e para baixo margin-bottom(margem-baixo). Caso tenha reparado os comandos são na pratica margem e o local dela em inglês, então caso tenha um conhecimento basico no idioma possivelmente tera uma facilidade a mais em memorizalos. Logo em seguida iremos ver cada um deles na pratica.
